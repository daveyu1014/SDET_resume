## 📌 Overview

This project is a lightweight, performance-optimized resume website designed to reflect my engineering mindset as a Software Development Engineer in Test (SDET).

The design focuses on:

- Clean architecture
- No external UI frameworks
- Accessibility & semantic structure
- Performance optimization
- Dark mode support
- Print-friendly layout (PDF ready)

---

## 🚀 Tech Stack

- HTML5 (Semantic Structure)
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript
- GitHub Pages (Deployment)

No external frameworks. No unnecessary dependencies.

---

## ✨ Features

- 🌙 Dark / Light Mode Toggle (with localStorage persistence)
- 📄 Downloadable Resume PDF
- 📱 Fully Responsive Design
- ⚡ Fast Loading (No heavy libraries)
- 🧱 Semantic HTML Structure
- 🔍 SEO-ready meta configuration

---

## 🧠 Engineering Decisions

### 1. No Framework Approach
The site intentionally avoids Bootstrap or other UI libraries to:

- Reduce bundle size
- Improve performance
- Demonstrate core frontend fundamentals

### 2. Dark Mode Implementation
Implemented using CSS custom properties and class toggling with persistent localStorage state.

### 3. GitHub Pages Deployment
Hosted using GitHub Pages for simplicity and reliability.
